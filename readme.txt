Configuration for this application

npm init to inititate node app
npm install express --save
npm install nodemon --save    -- will help autorestart the server after we save the changes.
npm install morgan --save     -- will help to log the incoming request.
npm install --save body-parser --  will help to parse or formate the incoming request in the nice manner.
To start the server go to the project folder and run -         node server.js
